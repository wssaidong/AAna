## 四、核心问题分析

### 问题1：大盘数据未获取，导致仓位判断完全失效
**现象：** 报告生成时4个指数数据全部显示"数据待获取"，导致大盘概览缺失。
**根因：** `fetch_stock_data.py` 在17:00生成报告时未能成功获取实时数据，或数据获取在报告生成之后才完成。
**影响：** 仓位建议50-70%与实际大盘环境（创业板-1.22%应为防守模式≤3成）严重不符。

### 问题2：新易盛标记"最佳买点"后继续暴跌4.43%
**现象：** 报告将新易盛（-4.43%）标记为"今日回调但未暴跌"的最佳买点，写入买点表。
**根因：** 评分系统只看当日跌幅-4.43%属于"大幅回调-7%~-3%"区间（+15分），忽略了：
- 该股前日（04-14）涨幅已超10%，高位追进风险极大
- 成交额超235亿创天量，高位放量是主力出货信号
- 报告未标注新易盛近期已有较大涨幅

### 问题3：海天瑞声高开低走，未触发高开过滤
**现象：** 海天瑞声开盘135.52（高开），收盘130.05（跌-2.36%），冲高回落形态。
**根因：** 指标体系中虽有"开盘涨幅>2.5%高开陷阱"规则，但筛选时未将此信号用于过滤已入选股票。

### 问题4：中际旭创、源杰科技等上涨股未被重点推荐
**现象：** 中际旭创（+0.90%）、源杰科技（+0.90%）在数据中表现良好，但不在Top10。
**根因：** 这些股票当日涨幅在0~3%区间，技术分计算时涨幅分偏低；而新易盛-4.43%反因"大幅回调"得了更高技术分，排名虚高。

### 问题5：热点板块判断部分准确但不够精细
**现象：** AI算力方向（寒武纪+2.58%、海光+1.29%）判断正确，但人形机器人板块整体偏弱。
**根因：** 板块判断以月为单位，未能精细到周级别轮动，且昊志机电、五洲新春在机器人线中表现不佳。

## 五、优化建议

### 建议1：修复大盘数据获取时序（高优先级）
**问题：** 报告生成时大盘数据缺失，导致仓位建议失准。
**方案：**
- 强制要求 `fetch_stock_data.py` 在报告生成前先执行并成功获取数据
- 增加数据有效性校验：若任一指数数据缺失，报告禁止生成或降级为"数据异常，仅供参考"
- 或将报告生成时间推迟至17:15，确保数据已就绪

### 建议2：修正"最佳买点"评分逻辑（高优先级）
**问题：** 新易盛被标记最佳买点后跌4.43%，"大幅回调"评分逻辑反成陷阱。
**方案（修改stock.md）：**
```
原规则：回调-7%~-3% → +15分（大幅回调）
改为：
  - 回调-7%~-3%，但近5日涨幅 > 10% → +5分（高位接刀，风险大于机会）
  - 回调-7%~-3%，且近5日涨幅 < 5% → +15分（超跌反弹，安全边际高）
```
同时增加：
```
新增风险信号：
  - 近5日涨幅 > 15%：即使当日回调也不推荐（可能是主力出货后的补跌）
  - 成交额创近20日天量：当日即使上涨也降权（主力借利好出货）
```

### 建议3：增加高开低走过滤规则（中优先级）
**问题：** 海天瑞声等高开低走股未被过滤。
**方案（修改stock.md）：**
```
在第四部分风险信号中增加：
  - 开盘价 > 收盘价 + 2%：高开低走，过滤（仙人指路形态）
  - 或在选股流程 Step 4.5 中增加：扫描候选股，若收盘价 < 开盘价 - 1.5%，剔除或降权
```

### 建议4：调整技术分涨幅计算权重（中优先级）
**问题：** 下跌-4.43%反而比上涨+0.90%技术分更高，逻辑反向。
**方案（修改stock.md）：**
```
技术分计算调整：
  - 温和下跌-3%~0%且近5日未大涨 → +10分（不追高，安全垫好）
  - 大幅下跌-7%~-3%且近5日涨幅 < 5% → +15分（超跌反弹）
  - 大幅下跌-7%~-3%但近5日涨幅 > 10% → +3分（高位接刀）
  - 温和上涨0~5% → +10分（趋势健康）
  - 大幅上涨>5% → +5分（已走强，不追）
```

### 建议5：增加"未推荐但表现好"的自检机制（低优先级）
**方案：** 复盘时增加对比项：数据中有但报告中未推荐的个股，标注"遗漏候选"供下次参考。
