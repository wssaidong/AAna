## 六、优化建议（stock.md调整）

### 建议1：涨停股不再简单降权（高优先级）
**现有规则：**
```
涨停>9%：-15分
```
**建议修改为：**
```
涨停+9%以上，且满足以下条件之一 → +10分（而非-15分）
  1. 股价创近20日新高（突破前高，看涨延续）
  2. 当日成交量 < 前日2倍（未过度放量，健康涨停）
  3. 近5日累计涨幅 < 15%（未连续涨停，安全边际）

涨停+9%以上，但以下情况 → -10分（保留风险提示）
  1. 近5日累计涨幅 > 20%（连续涨停过热）
  2. 成交量创近20日天量（主力借利好出货嫌疑）
```

### 建议2：增加"趋势确认型"加分项（高优先级）
**新增指标：**
```
连续2日涨幅>5%，且近5日涨幅<15% → +10分
（视为趋势确认信号，积极看多）
```

### 建议3：中性模式大盘股降权（中优先级）
**中性模式（仓位3-5成）下，增加：**
```
流通市值>100亿的个股 → 综合评分×0.85
（大盘股弹性不足，中性市场下优先级降低）
```

### 建议4：热点板块覆盖扩展（中优先级）
**AI算力/DS概念板块覆盖范围扩大：**
```
原范围：AI芯片、算力服务器
建议新增：LED驱动芯片（삼安光电类）、化合物半导体、光通信器件
```

### 建议5：确保大盘数据实时性（高优先级）
**选股报告生成前置检查：**
