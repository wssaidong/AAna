## 五、核心问题分析

### 问题1：选股逻辑鼓励"追高"——最高评分股票反而表现最差

报告的"回调买点"逻辑存在自我矛盾：
- 新易盛（评分71，最高）：开盘+6.6% → 收盘-1.87%，高开低走
- 五洲新春（评分70）：开盘+4.2% → 收盘-1.59%，高开低走
- 中微公司（评分70）：开盘+0.8% → 收盘-2.29%，高开低走

**原因：** 评分系统给"回调-3%~0%"加12分，给"温和上涨0~5%"加10分，但无法区分"高开后回落"与"真正回调"。当日大幅高开（+4%~+6%）的股票反而得分高，但这类股票次日或当日尾盘往往回落。

### 问题2：高位放量股被推荐 → 实为"主力出货"

- 新易盛开盘涨6.6%，成交额191亿（极高），属于典型的高位放量
- 中微公司开盘涨0.8%，成交额298亿，放量下跌
- 中际旭创开盘涨6%，成交额1673亿

这些都是高位放量阴线形态，是stock.md明确要求过滤的风险信号，但未被执行。

### 问题3：数据获取时机问题

报告在17:00生成时使用收盘价，而涨跌幅描述的是"早盘"（9:00数据），造成基准时间混乱：
- 早盘涨跌幅 vs 收盘涨跌幅定义不一致
- 无法准确判断"最佳买点"

### 问题4：黑马板块（储能/绿电）被边缘化

储能板块今日实际表现强劲（固德威+4.54%、宁德时代+2.68%），但：
- 仅作为次要板块列在报告末尾
- 宁德时代/比亚迪等大票未被正式推荐
- 源杰科技（+6.20%）完全未被提及

### 问题5：评分对成交额的利用不足

海光信息（+5.79%）成交额1057亿，说明大资金参与，但综合评分仅63；
新易盛成交额1911亿，明显过大（可能主力已出货），评分却给到71。

## 六、stock.md v2.1 优化内容

本次复盘触发以下三处核心修改：

### 1. 风险信号新增三条高开过滤
- 开盘涨幅 > 2.5% → 过滤（高开过多，尾盘回落概率大）
- 开盘价 > 3日内最高价 → 过滤（价格已透支空间）
- 成交量创近期天量但价格不涨 → 过滤（主力出货嫌疑）

### 2. 新增指标18：主力净流入确认型
```
条件：5日大单净流入 > 0 + 成交额5~100亿 + 涨幅2~6% + 
      收盘价在区间中位以上 + 近20日涨幅 < 30%
```
核心逻辑：拒绝高开低走（收盘在中位以上），确认资金持续入场（5日净流入）。

### 3. 新增Step 4.5：收盘前30分钟确认
```
收盘前30分钟扫描候选股：
- 若股价高于开盘价 → 保留
- 若股价低于开盘价且跌幅>1% → 剔除或降权
- 若成交额创天量但涨幅<1% → 降权或剔除
```
